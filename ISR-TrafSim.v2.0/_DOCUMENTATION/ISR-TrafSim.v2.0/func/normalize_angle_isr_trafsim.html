-DESCRIPTION--------------------------------------------------------------

-USE----------------------------------------------------------------------

 -> Input(s)
   » ang   -   Input angle
   » in    -   Type of input angle ('rad'/'deg')
   » out   -   Type of output angle ('rad'/'deg')

       If 2 convert ang to -180<ang<180

 -> Output(s)
   » ang   -   Angle converted according to specifications
